# Button - 按钮
常用的操作按钮

## 基础用法
基础的按钮用法。

<ClientOnly>
  <button-demo type="basic"/>
</ClientOnly>

## 禁用状态
按钮不可用状态。

<ClientOnly>
  <button-demo type="disabled"/>
</ClientOnly>

## 图标按钮
带图标的按钮可增强辨识度（有文字）或节省空间（无文字）。

<ClientOnly>
  <button-demo type="icon"/>
</ClientOnly>

## 按钮组
以按钮组的方式出现，常用于多项类似操作。

<ClientOnly>
  <button-demo type="group"/>
</ClientOnly>

## 加载中
点击按钮后进行数据加载操作，在按钮上显示加载状态。

<ClientOnly>
  <button-demo type="loading"/>
</ClientOnly>
